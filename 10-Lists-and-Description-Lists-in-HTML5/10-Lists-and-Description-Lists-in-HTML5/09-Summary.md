# HTML List Elements

The following HTML elements are used to create lists:

| Element | Purpose |
|---------|---------|
| `<ol>` | Creates an ordered (numbered) list. |
| `<ul>` | Creates an unordered (bulleted) list. |
| `<li>` | Represents an individual list item. |
| `<dl>` | Creates a description list. |
| `<dt>` | Defines a term within a description list. |
| `<dd>` | Defines the description of a term. |

# Best Practices Recap

When working with HTML lists, always remember to:

- Choose the appropriate list type.
- Use semantic HTML elements.
- Keep list items concise and meaningful.
- Maintain proper indentation.
- Avoid unnecessary nesting.
- Use CSS for presentation instead of HTML attributes where appropriate.
- Ensure accessibility for all users.
- Keep styling consistent across the website.

---

# Common Mistakes to Avoid

Avoid the following practices:

- Using ordered lists when sequence is unimportant.
- Using unordered lists for numbered instructions.
- Misusing description lists for unrelated content.
- Manually typing numbers instead of using `<ol>`.
- Omitting the `<li>` element.
- Creating deeply nested list structures unnecessarily.
- Using lists solely for page layout.
- Ignoring semantic HTML principles.
