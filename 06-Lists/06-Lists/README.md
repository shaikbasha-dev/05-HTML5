# HTML5 Lists

## Overview

Lists are one of the fundamental building blocks of HTML documents. They are used to organize related information in a structured and readable format. HTML5 provides three types of lists:

- Ordered Lists (`<ol>`)
- Unordered Lists (`<ul>`)
- Description Lists (`<dl>`)

Lists are widely used for navigation menus, feature lists, step-by-step instructions, FAQs, documentation, product specifications, and many other real-world applications.

This module provides a comprehensive understanding of HTML5 Lists, covering syntax, attributes, examples, best practices, interview questions, and practical source code examples.
