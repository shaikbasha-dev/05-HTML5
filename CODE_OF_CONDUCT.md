# Code of Conduct

## Our Commitment

The **05-HTML5** repository is committed to providing a welcoming, inclusive, respectful, and collaborative environment for everyone. Whether you are a beginner, student, educator, or experienced developer, your participation is valued and appreciated.

Our goal is to foster a positive learning community where knowledge can be shared openly and respectfully.

---

# Purpose

This Code of Conduct establishes expectations for behavior within this repository and helps maintain a professional, friendly, and inclusive environment for all contributors and learners.

---

# Our Standards

Examples of behavior that contribute to a positive environment include:

- Being respectful and considerate.
- Using welcoming and inclusive language.
- Accepting constructive feedback gracefully.
- Sharing knowledge to help others learn.
- Encouraging collaboration.
- Respecting differing viewpoints and experiences.
- Writing clear, professional, and educational content.
- Reporting issues responsibly.

---

# Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment or discrimination of any kind.
- Offensive, abusive, or insulting language.
- Personal attacks or intimidation.
- Spam or promotional content unrelated to the repository.
- Sharing confidential or private information without permission.
- Intentionally submitting misleading or harmful content.
- Disruptive behavior that negatively affects the learning environment.

---

# Responsibilities

Repository maintainers are responsible for:

- Maintaining a respectful environment.
- Reviewing contributions fairly.
- Providing constructive feedback.
- Enforcing this Code of Conduct when necessary.
- Making decisions that benefit the educational goals of the repository.

---

# Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Repository documentation
- Source code contributions
- Community interactions related to this repository

---

# Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it by opening a GitHub Issue or contacting the repository maintainer.

When reporting an issue, include:

- A clear description of the incident.
- Relevant screenshots or evidence (if applicable).
- The date and context of the incident.
- Any additional information that may help resolve the issue.

All reports will be handled respectfully and confidentially whenever possible.

---

# Enforcement

Maintainers may take appropriate action in response to unacceptable behavior, including:

- Providing a warning.
- Requesting changes.
- Removing inappropriate content.
- Rejecting contributions.
- Restricting participation in severe or repeated cases.

---
