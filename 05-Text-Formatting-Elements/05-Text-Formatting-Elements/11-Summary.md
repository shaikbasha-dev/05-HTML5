# Summary - HTML Text Formatting Elements

## Module Overview

Text formatting elements are fundamental components of HTML5 that allow developers to present content in a meaningful, readable, and semantically correct manner. While some elements primarily affect the visual appearance of text, many HTML5 formatting elements also convey semantic meaning, improving accessibility, search engine optimization (SEO), and overall document structure.

This module introduced the most commonly used HTML text formatting elements, their purposes, syntax, practical examples, best practices, and real-world applications.

---

# Topics Covered

During this module, the following concepts were explored:

- Introduction to HTML Text Formatting
- Bold (`<b>`) and Strong (`<strong>`) Text
- Italic (`<i>`) and Emphasized (`<em>`) Text
- Underlined (`<u>`) and Marked (`<mark>`) Text
- Small (`<small>`), Deleted (`<del>`), and Inserted (`<ins>`) Text
- Subscript (`<sub>`) and Superscript (`<sup>`) Text

- HTML5 Text Formatting Best Practices
- Interview Questions and Answers

# Best Practices Recap

- Prefer semantic HTML elements over purely visual formatting.
- Use `<strong>` instead of `<b>` when indicating importance.
- Use `<em>` instead of `<i>` for emphasis.
- Use `<mark>` only for contextually relevant content.
- Use `<code>` and `<pre>` for programming examples.
- Provide the `title` attribute with `<abbr>` whenever appropriate.
- Keep HTML code clean, properly indented, and easy to read.
- Use formatting elements according to their intended purpose.
