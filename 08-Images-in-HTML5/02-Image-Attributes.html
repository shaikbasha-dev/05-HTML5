HTML5 Image Attributes

1. src Attribute

The src attribute specifies the location of the image.

2. alt Attribute

The alt attribute provides alternative text if the image cannot be displayed.

Summary of Image Attributes

Attribute Purpose
src Specifies the image file location.
alt Provides alternative text.
width Sets image width.
height Sets image height.
title Displays a tooltip.
loading Controls when the image is loaded.
