HTML5 Image Paths

1. Image in the Same Folder

The image file is stored in the same directory as this HTML file.

src="logo.png"

2. Image Inside an Images Folder

The image is stored inside an images folder.

src="images/nature.jpg"

3. Image from Parent Folder

The image is located one directory above the current HTML file.

src="../images/banner.jpg"

4. Absolute Image Path

The image is loaded from an external website using a complete URL.

src="https://via.placeholder.com/400x200"

Relative vs Absolute Paths

Relative Path Absolute Path
Used for project images. Used for externally hosted images.
Portable. Depends on external servers.
No domain name required. Includes protocol and domain.
Ideal for local development. Ideal for CDN or third-party resources.

Best Practices

  • Use relative paths for project resources.
  • Organize images inside dedicated folders.
  • Use meaningful file names.
  • Always verify image paths before deployment.
  • Use HTTPS for external image resources.
  • Always include the alt attribute.
