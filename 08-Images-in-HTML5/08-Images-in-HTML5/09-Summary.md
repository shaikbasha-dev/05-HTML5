# Image Formats at a Glance

| Format | Best Used For | Transparency | Animation |
|---------|---------------|--------------|-----------|
| JPEG | Photographs | No | No |
| PNG | Logos, Icons, Screenshots | Yes | No |
| GIF | Simple Animations | Limited | Yes |
| SVG | Logos, Icons, Vector Graphics | Yes | No |
| WebP | Modern Responsive Websites | Yes | Yes |

# Image Attributes Summary

| Attribute | Purpose |
|-----------|---------|
| `src` | Specifies the image source. |
| `alt` | Provides alternative text for accessibility. |
| `width` | Sets the image width. |
| `height` | Sets the image height. |
| `title` | Displays tooltip text. |
| `loading` | Controls image loading behavior. |
| `srcset` | Specifies multiple image sources. |
| `sizes` | Defines the expected display size for responsive images. |

---

# Relative vs Absolute Paths

| Relative Path | Absolute Path |
|---------------|---------------|
| Used within the project | Used for external resources |
| Portable | Depends on external URLs |
| Faster for local assets | Requires internet access |
| Easier to maintain | Suitable for CDNs and hosted images |

# Responsive Image Techniques

Modern HTML5 provides several approaches for responsive images:

- CSS (`max-width: 100%`)
- `<picture>` element
- `srcset` attribute
- `sizes` attribute
- Responsive layouts using CSS media queries

These techniques help deliver optimized images across desktops, tablets, and smartphones.

---

# Best Practices Checklist

Before publishing a webpage, ensure that:

- Images use the appropriate file format.
- Every image includes a meaningful `alt` attribute.
- Images are optimized for web use.
- Image dimensions are specified when appropriate.
- Responsive image techniques are implemented.
- Lazy loading is used for non-critical images.
- Images are stored in organized folders.
- File names are descriptive and meaningful.
- External image URLs use HTTPS.
- All image links have been tested.

---

# Common Mistakes to Avoid

Avoid these common issues:

- Missing `alt` attributes.
- Using oversized images.
- Choosing the wrong image format.
- Using broken image paths.
- Distorting images with incorrect dimensions.
- Ignoring responsive design.
- Failing to optimize images before uploading.
- Using generic or unclear file names.
