HTML5 Basic Video Example

This example demonstrates how to embed a video using the HTML5 <video> element.

Explanation

  • <video> creates a video player.
  • controls displays the browser's built-in playback controls.
  • width specifies the width of the video player.
  • <source> specifies the video file.
  • type specifies the MIME type of the video.
  • The fallback text appears if the browser does not support HTML5 video.
