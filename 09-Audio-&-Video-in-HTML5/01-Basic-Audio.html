HTML5 Basic Audio Example

This example demonstrates how to embed an audio file using the HTML5 <audio> element.

Explanation

  • <audio> creates an audio player.
  • controls displays the browser's built-in playback controls.
  • <source> specifies the audio file.
  • type specifies the MIME type of the audio.
  • The fallback text appears if the browser does not support HTML5 audio.
