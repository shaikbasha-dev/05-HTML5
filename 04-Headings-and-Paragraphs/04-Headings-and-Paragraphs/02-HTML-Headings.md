# HTML Headings

## Introduction

HTML headings are used to define the titles and subtitles of a webpage. They help organize content into logical sections, making webpages easier to read, understand, and navigate. Headings also play an important role in accessibility and Search Engine Optimization (SEO) by providing a clear structure for both users and search engines.

HTML provides six heading levels, ranging from `<h1>` (the highest level of importance) to `<h6>` (the lowest level of importance).

---

# Definition

An **HTML Heading** is an element used to define the title or heading of a section within a webpage.

Headings establish the hierarchical structure of webpage content and improve readability.

---

# HTML Heading Elements

HTML provides six heading elements:

```html
<h1>Heading Level 1</h1>

<h2>Heading Level 2</h2>

<h3>Heading Level 3</h3>

<h4>Heading Level 4</h4>

<h5>Heading Level 5</h5>

<h6>Heading Level 6</h6>
```

---

# Syntax

General syntax:

```html
<h1>Heading Text</h1>
```

Example:

```html
<h2>About HTML5</h2>
```

---

# Heading Levels

## `<h1>`

- Highest level heading.
- Represents the main title of the webpage.
- Typically used only once per page.

Example:

```html
<h1>HTML Tutorial</h1>
```

---

## `<h2>`

- Represents major sections of the webpage.
- Used for primary headings under the main title.

Example:

```html
<h2>Introduction</h2>
```

---

## `<h3>`

- Represents subsections within an `<h2>` section.

Example:

```html
<h3>HTML Elements</h3>
```

---

## `<h4>`

- Used for subtopics within an `<h3>` section.

Example:

```html
<h4>Heading Syntax</h4>
```

---

## `<h5>`

- Represents lower-level subsections.

Example:

```html
<h5>Additional Information</h5>
```

---

## `<h6>`

- Lowest heading level.
- Used for highly specific information.

Example:

```html
<h6>Reference Notes</h6>
```

---

# Complete Example

```html
<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <title>HTML Headings</title>

</head>

<body>

    <h1>HTML Tutorial</h1>

    <h2>Introduction</h2>

    <h3>HTML Basics</h3>

    <h4>HTML Elements</h4>

    <h5>Examples</h5>

    <h6>Summary</h6>

</body>

</html>
```

# Importance of HTML Headings

HTML headings:

- Organize webpage content.
- Improve readability.
- Create logical document structure.
- Enhance accessibility.
- Improve Search Engine Optimization (SEO).
- Help screen readers navigate webpages.
- Improve user experience.

---

# Advantages

- Creates structured webpages.
- Makes content easier to understand.
- Improves accessibility.
- Supports semantic HTML.
- Enhances SEO.
- Simplifies webpage navigation.

---

# Best Practices

- Use only one `<h1>` for the main page title.
- Follow heading levels sequentially.
- Avoid skipping heading levels.
- Write clear and descriptive headings.
- Keep headings concise.
- Use headings to organize content logically.

---

# Common Mistakes

- Using multiple `<h1>` elements unnecessarily.
- Skipping heading levels (for example, `<h1>` directly to `<h4>`).
- Using headings only for visual styling.
- Writing vague or unclear headings.
- Choosing heading levels based solely on font size.

---

# Interview Questions

### 1. What are HTML headings?

HTML headings define the titles and section headings of a webpage.

---

### 2. How many heading levels are available in HTML?

HTML provides six heading levels, from `<h1>` to `<h6>`.

---

### 3. Which heading has the highest importance?

`<h1>`.

---

### 4. Why are HTML headings important?

They improve webpage organization, accessibility, readability, navigation, and SEO.

---

### 5. Should heading levels be skipped?

No. Heading levels should follow a logical hierarchy whenever possible.

# Summary

HTML headings provide the structural framework for webpage content. They establish a logical hierarchy, improve readability, support accessibility, and help search engines understand the organization of information. Proper use of heading elements is a fundamental best practice in HTML5 and an essential skill for every web developer.
