# Tables in HTML5

## Module Overview

HTML tables are used to organize and display data in a structured format consisting of rows and columns. They are widely used for presenting tabular information such as student records, employee details, product catalogs, financial reports, schedules, invoices, and many other types of structured data.

HTML5 provides semantic table elements that improve readability, accessibility, maintainability, and search engine optimization. By combining table elements with CSS, developers can create responsive, professional, and visually appealing tables for modern web applications.

This module provides a complete understanding of HTML5 tables, including their structure, elements, attributes, table sections, spanning techniques, accessibility considerations, best practices, and practical implementation examples.

# Learning Objectives

After completing this module, you will be able to:

- Understand the purpose of HTML tables.
- Create tables using the `<table>` element.
- Create rows using the `<tr>` element.

- Create header cells using the `<th>` element.
- Add table captions using the `<caption>` element.
- Organize tables using `<thead>`, `<tbody>`, and `<tfoot>`.
- Merge rows using the `rowspan` attribute.
- Merge columns using the `colspan` attribute.
- Build responsive HTML tables.
- Apply HTML5 table best practices.
- Prepare for HTML table interview questions.
