# HTML Elements Learned

| Element | Purpose |
|----------|---------|
| `<table>` | Creates the main table container. |
| `<tr>` | Defines a table row. |
| `<th>` | Defines a table header cell. |
| `<td>` | Defines a table data cell. |
| `<caption>` | Provides a title for the table. |
| `<thead>` | Groups header rows. |
| `<tbody>` | Groups the main table data. |
| `<tfoot>` | Groups footer rows. |

---

# Attributes Learned

| Attribute | Purpose |
|-----------|---------|
| `colspan` | Merges multiple columns into one cell. |
| `rowspan` | Merges multiple rows into one cell. |
| `scope` | Associates header cells with rows or columns for accessibility. |

# Best Practices Checklist

Use this checklist while creating HTML tables:

- ✅ Use tables only for tabular data.
- ✅ Use semantic table elements.
- ✅ Add meaningful captions.
- ✅ Use `<th>` for headers.
- ✅ Apply the `scope` attribute where appropriate.
- ✅ Keep table structures consistent.
- ✅ Use CSS instead of deprecated HTML attributes.
- ✅ Make tables responsive.
- ✅ Test tables on multiple browsers and devices.
- ✅ Follow accessibility guidelines.

---

# Common Mistakes to Avoid

Avoid these common mistakes:

- Using tables for page layouts.
- Ignoring semantic elements.
- Omitting header cells.
- Forgetting captions.
- Overusing `rowspan` and `colspan`.
- Using deprecated HTML attributes.
- Ignoring responsive design.
- Creating overly complex tables.
