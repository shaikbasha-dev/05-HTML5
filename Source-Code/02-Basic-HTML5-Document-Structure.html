Basic HTML5 Document Structure

Every HTML5 webpage follows a standard document structure that helps web browsers understand and render the content correctly.

Main Parts of an HTML5 Document

  1. <!DOCTYPE html> — Declares the document as an HTML5 document.
  2. <html> — Root element that contains the entire webpage.
  3. <head> — Stores metadata, title, styles, and external resources.
  4. <body> — Contains all the visible content displayed in the browser.

Understanding the HTML5 document structure is essential before learning individual HTML elements and building web pages.
