HTML5 Text Formatting Elements

HTML5 provides several text formatting elements to emphasize, highlight, underline, strike through, or display text with special meaning.

1. Bold Text

This text is displayed in bold using the <b> tag.

2. Strong Text

This text indicates strong importance using the <strong> tag.

3. Italic Text

This text is displayed in italic using the <i> tag.

4. Emphasized Text

This text is emphasized using the <em> tag.

5. Underlined Text

This text is underlined using the <u> tag.

6. Highlighted Text

This is highlighted text using the <mark> tag.

7. Small Text

This text is displayed in a smaller font size.

8. Deleted Text

The original price was ₹1,000.

9. Inserted Text

The updated price is ₹800.

10. Subscript Text

Water is represented as H2O.

11. Superscript Text

Einstein's famous equation is E = mc2.

Best Practices

  • Use <strong> for important content instead of <b>.
  • Use <em> for emphasized text instead of <i>.
  • Use formatting tags only when they add meaning to the content.
  • Avoid excessive formatting as it reduces readability.
  • Choose semantic elements whenever possible.
