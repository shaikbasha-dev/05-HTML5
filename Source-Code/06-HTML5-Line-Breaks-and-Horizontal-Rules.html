HTML5 Line Breaks and Horizontal Rules

HTML provides the <br> tag to insert a line break and the <hr> tag to create a thematic break (horizontal line) between sections of content.

Best Practices

  • Use <br> only for line breaks such as poems, addresses, or contact information.
  • Avoid using multiple <br> tags to create spacing between elements.
  • Use CSS for layout and spacing instead of repeated line breaks.
  • Use <hr> to separate related sections of content.
  • Keep webpage content clean, readable, and semantically meaningful.
