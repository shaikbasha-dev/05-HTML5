HTML5 Favicon

A favicon (Favorite Icon) is the small icon displayed in the browser tab, bookmarks, browser history, and shortcuts. It helps users quickly identify a website.

Basic Favicon Example

The favicon for this webpage is added using the <link> element inside the <head> section.

HTML Code Used

<link rel="icon"
      type="image/png"
      href="images/favicon.png">

Supported Favicon Formats

  • .ico (Recommended for maximum browser compatibility)
  • .png (Most commonly used)
  • .svg (Scalable Vector Graphic)
  • .gif
  • .jpg / .jpeg (Less common)

Where Favicons Are Displayed

  • Browser Tabs
  • Bookmarks
  • Browser History
  • Mobile Home Screen Shortcuts
  • Search Results (Some Browsers)

Best Practices

  • Use a square image (16×16, 32×32, or 48×48 pixels).
  • Prefer PNG or ICO formats.
  • Keep the icon simple and recognizable.
  • Store favicon files in a dedicated images or assets folder.
  • Always define the favicon inside the <head> section.
