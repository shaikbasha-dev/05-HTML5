HTML5 Description Lists

HTML5 provides description lists to display terms and their corresponding descriptions. A description list is created using the <dl> element, while <dt> defines the term and <dd> defines its description.

Best Practices

  • Use description lists for definitions, glossaries, FAQs, and metadata.
  • Place every term inside the <dt> element.
  • Place each explanation inside the <dd> element.
  • Keep descriptions clear, concise, and meaningful.
  • Use semantic HTML elements to improve accessibility and readability.
