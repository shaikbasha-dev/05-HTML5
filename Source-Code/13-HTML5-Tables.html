HTML5 Tables

HTML5 tables are used to display data in rows and columns. A table is created using the <table> element, where rows are defined using <tr>, header cells using <th>, and data cells using <td>.

HTML Table Elements

  • <table> - Creates a table.
  • <caption> - Provides a title for the table.
  • <thead> - Defines the table header section.
  • <tbody> - Defines the main table content.
  • <tfoot> - Defines the footer section.
  • <tr> - Defines a table row.
  • <th> - Defines a table heading cell.
  • <td> - Defines a table data cell.

Best Practices

  • Use <caption> to provide a meaningful title.
  • Separate header, body, and footer using semantic table elements.
  • Use <th> for headings instead of <td>.
  • Keep tables well-organized and easy to read.
  • Use CSS for styling instead of the border attribute in production websites.
