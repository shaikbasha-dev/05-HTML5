Internal and External Links in HTML5

Internal Links

Internal links connect one page to another page within the same website.

  • Home
  • About Us
  • Services
  • Contact Us

External Links

External links connect to resources located on different websites.

  • Google
  • GitHub
  • MDN Web Docs
  • W3C

Comparison

Internal Link External Link
Links pages within the same website. Links to another website.
Usually uses Relative URLs. Usually uses Absolute URLs.
Improves website navigation. Provides access to external resources.

Summary

Internal links help users navigate within the same website, while external links allow users to visit resources hosted on other domains.
