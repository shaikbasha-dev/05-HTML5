# Basic Structure of HTML5

## Overview

The Basic Structure of HTML5 is the foundation of every web page. Every HTML document follows a standard structure that helps browsers correctly interpret and render web content.

Understanding the HTML5 document structure is one of the most important skills for beginners because every webpage, regardless of its complexity, starts with the same basic layout.

This module explains each part of the HTML5 document, including the DOCTYPE declaration, HTML element, HEAD section, BODY section, metadata, title element, language attribute, proper nesting, indentation, and coding best practices.

---

# Learning Objectives

After completing this module, you will be able to:

- Understand the standard HTML5 document structure.
- Explain the purpose of every structural element.
- Write a valid HTML5 document from scratch.
- Organize HTML code using proper nesting and indentation.
- Apply HTML5 best practices.
- Build clean, readable, and maintainable web pages.

# Best Practices

- Always begin with the HTML5 DOCTYPE declaration.
- Specify the document language using the `lang` attribute.
- Use UTF-8 character encoding.
- Configure the viewport for responsive design.
- Write meaningful page titles.
- Keep HTML properly indented.
- Follow semantic HTML principles.
- Write clean and maintainable code.
