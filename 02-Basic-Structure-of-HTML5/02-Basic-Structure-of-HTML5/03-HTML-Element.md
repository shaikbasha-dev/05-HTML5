# HTML Element

## Introduction

The `<html>` element is the root element of every HTML document. It serves as the container for all other HTML elements except the DOCTYPE declaration. Every webpage begins with the opening `<html>` tag and ends with the closing `</html>` tag.

The HTML element establishes the document as an HTML page and contains two major sections: the `<head>` element and the `<body>` element.

---

# Definition

The `<html>` element is the root element of an HTML document that encloses all the content and metadata of a webpage.

---

# Syntax

```html
<html lang="en">

    <head>
        ...
    </head>

    <body>
        ...
    </body>

</html>
```

---

# Purpose of the HTML Element

The HTML element is responsible for:

- Defining the root of the HTML document.
- Containing all webpage elements.
- Organizing the HEAD and BODY sections.
- Specifying the document language.
- Helping browsers correctly interpret the webpage.
- Improving accessibility and SEO.

---

# Attributes of the HTML Element

The most commonly used attribute is:

| Attribute | Description |
|-----------|-------------|
| `lang` | Specifies the language of the document. |

Example:

```html
<html lang="en">
```

Other examples:

```html
<html lang="fr">
```

```html
<html lang="es">
```

```html
<html lang="hi">
```

---

# HTML Element Structure

```html
<!DOCTYPE html>

<html lang="en">

<head>

    <title>My Webpage</title>

</head>

<body>

    <h1>Welcome</h1>

    <p>This is an HTML5 document.</p>

</body>

</html>
```

---

# Why is the HTML Element Important?

The HTML element:

- Acts as the root container.
- Defines the document language.
- Improves browser compatibility.
- Supports accessibility tools.
- Helps search engines understand webpage language.
- Provides a structured foundation for all HTML content.

---

# Advantages

- Organizes the webpage structure.
- Supports multilingual websites.
- Improves accessibility.
- Enhances SEO.
- Ensures proper browser rendering.
- Makes HTML documents standards-compliant.

---

# Best Practices

- Always include the `<html>` element.
- Specify the `lang` attribute.
- Place only the `<head>` and `<body>` elements directly inside the `<html>` element.
- Maintain proper indentation.
- Close the HTML element correctly.

---

# Common Mistakes

- Forgetting the `lang` attribute.
- Omitting the closing `</html>` tag.
- Placing content outside the HTML element.
- Incorrect nesting of HEAD and BODY elements.
- Poor indentation.
