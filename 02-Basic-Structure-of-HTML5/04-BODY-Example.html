HTML BODY Element Example

The BODY element contains all the visible content that users see and interact with in a web browser.

Visible Elements Inside the BODY

  • Headings
  • Paragraphs
  • Images
  • Hyperlinks
  • Lists
  • Tables
  • Forms
  • Audio and Video

Example Content

HTML5 provides numerous elements that can be placed inside the BODY section to create informative and interactive webpages.
